# GamerLog
Plataforma para registro e acompanhamento de jogatinas.

________________________________________

# Como usar
Basta criar sua conta, fazer login e começar a registrar suas jogatinas.

_______________________________________

# Detalhes técnicos

Desenvolvido em .NET e SQL Server, com dados dos jogos vindos da API do Rawg, cadastrados em uma tabela de jogos automaticamente, com dados complementados pelo Google Gemini.
